Windows 注册表笔记

发布于 2019-05-30 | 标签: Windows 、 Windows 7 、 Registry | 1分钟 | 121字数 | 浏览量::

记录遇到使用过的 Windows 注册表相关

问题类

  • 描述
    某个软件，不能选择应用为默认打开程序。例如 .jar 文件，默认使用 jd-gui.exe 打开。选择默认程序窗口中，浏览 jd-gui.exe 并没有在列表中出现。

  • 原因
    原有使用 jd-gui.exe 打开，但程序路径发生改变，注册表保存了原有的路径。

  • 解决
    删除注册表

HKEY_CURRENT_USER\Software\Classes\Applications\jd-gui.exe
